# matematik-oyin
🚀 Loyiha haqida Dastur tasodifiy sonlar generatsiyasi (rand() funksiyasi) va takrorlanuvchi operatorlar (loops) yordamida ishlaydi. Foydalanuvchi turli darajadagi matematik misollarni yechishi kerak.
✨ Asosiy xususiyatlari:
Tasodifiy savollar: Har safar yangi va turlicha misollar hosil qilinadi.

Darajalar tizimi: O'yinda qiyinchilik darajalari mavjud (masalan: oson, o'rta, qiyin).

Natijalarni hisoblash: To'g'ri va noto'g'ri javoblar soni kuzatib boriladi.

🛠 Texnologiyalar:
Til: C

Asboblar: Loops, rand(), scanf, printf
